Platform

Platform API

A REST service that makes all kostrack data available over HTTP — so finance teams, operations, and non-Python services can query spend, manage budgets, and pull data into Excel or BI tools without writing any code.

Who this is for

The Platform API lets any tool or service query AI spend data and enforce budgets over HTTP — without installing Python or the kostrack SDK. This is how you connect Excel, Power BI, a Node.js service, or a simple browser query directly to your AI governance data.

What it is

When you run docker compose up -d, a fourth container starts automatically alongside TimescaleDB, Grafana, and the SDK. It listens on port 8080 and exposes all governance data as a REST API — structured data you can read with any HTTP client.

The interactive API explorer (Swagger UI) is always available at http://localhost:8080/docs. You can run queries directly from the browser, fill in fields, and see live responses — no code required.

Checking it is running

# In a browser, visit:
http://localhost:8080/health/live

# You should see:
{"status": "ok", "service": "kostrack-api", "version": "0.2.0"}

Authentication

If your team has set a KOSTRACK_API_KEY in the server configuration, include it in every request as a bearer token. If no key is set (common for internal / local deployments), requests work without any authentication.

# Without auth (internal network, no key set)
curl http://localhost:8080/spend

# With auth key
curl -H "Authorization: Bearer your-api-key" http://localhost:8080/spend

What you can query

Spend data

Returns a breakdown of AI spending by provider, model, project, and day. All numbers are in USD.

# All spend, last 30 days
GET http://localhost:8080/spend

# Filter by project, last 7 days, by hour
GET http://localhost:8080/spend?project=openmanagr&days=7&granularity=hour

# Filter by provider
GET http://localhost:8080/spend?provider=anthropic&days=30

Example response:

{
  "total_cost_usd": 1.1752,
  "filters": { "days": 30, "project": "openmanagr" },
  "rows": [
    {
      "bucket": "2026-03-22T00:00:00",
      "provider": "anthropic",
      "model": "claude-sonnet-4-6",
      "project": "openmanagr",
      "feature": "invoice-extraction",
      "calls": 142,
      "cost_usd": 0.8512
    }
  ]
}

Workflow trace cost breakdown

Returns the full cost breakdown for a single multi-step AI workflow run, split by which AI model was responsible for what percentage of the cost.

GET http://localhost:8080/spend/trace/11111111-1111-1111-1111-111111111111
{
  "trace_id": "11111111-...",
  "total_cost_usd": 0.02,
  "call_count": 3,
  "model_breakdown": [
    { "model": "deepseek-reasoner",  "cost_usd": 0.0142, "pct": 71.0 },
    { "model": "claude-sonnet-4-6",  "cost_usd": 0.0058, "pct": 29.0 }
  ]
}

Budgets

List all budgets with their current period spend and status.

GET http://localhost:8080/budgets
{
  "budgets": [
    {
      "tag_key": "project",
      "tag_value": "openmanagr",
      "period": "monthly",
      "limit_usd": 50.0,
      "spent_usd": 42.10,
      "pct": 84.2,
      "enforce": false,
      "status": "alert"
    }
  ]
}

Create a new budget:

POST http://localhost:8080/budgets
Content-Type: application/json

{
  "tag_key": "project",
  "tag_value": "openmanagr",
  "period": "monthly",
  "limit_usd": 50.0,
  "alert_threshold": 0.80,
  "enforce": true
}

Models

All AI models ever used, with total call count and lifetime spend.

GET http://localhost:8080/models

Pricing

# View current pricing for all models
GET http://localhost:8080/pricing

# Sync bundled pricing into the database
POST http://localhost:8080/pricing/sync

Budget enforcement from any service

The /budgets/check endpoint is designed for pre-call enforcement from any language. Send your tags before making an AI call. Get back budget states. If a hard budget is exceeded, the API returns 402 Payment Required — your service treats this as a signal to stop the call.

POST http://localhost:8080/budgets/check
Content-Type: application/json

{
  "tags": { "project": "openmanagr", "team": "engineering" },
  "estimated_cost": 0.01
}

# Returns 200 if under budget (with any warnings)
# Returns 402 if a hard-limit budget is exceeded

In Node.js:

const res = await fetch('http://kostrack:8080/budgets/check', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    tags: { project: 'openmanagr' },
    estimated_cost: 0.01,
  }),
});

if (res.status === 402) {
  throw new Error('Budget exceeded — AI call blocked');
}
// Proceed with AI call

Using with Excel or Power BI

From Excel, use Data → Get Data → From Web and enter:

http://localhost:8080/spend?days=30

Excel will load the JSON response and let you transform it into a table. You can set it to refresh on a schedule so your spend report is always current without anyone exporting a CSV.

GET/budgetsAll budgets with live period spend

DELETE/budgets/{id}Delete a budget
POST/budgets/checkPre-call enforcement — 402 if exceeded
GET/modelsAll models with lifetime stats
GET/pricingActive pricing for all models
POST/pricing/syncSync bundled pricing to database
Interactive API explorer

The full API with live request / response forms is always available at http://localhost:8080/docs. You can try every endpoint directly in the browser, see the exact request format, and read the response — no code or curl required.
