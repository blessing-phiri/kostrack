Platform API

A REST service that exposes all kostrack governance primitives over HTTP — so any language, team, or service can query spend, manage budgets, and enforce limits without the Python SDK.

What it is

The Platform API is a thin FastAPI service that exposes all kostrack governance primitives over HTTP. It ships as a fourth container alongside TimescaleDB, Grafana, and the SDK — and starts automatically with docker compose up -d.

Any service in any language can query spend, manage budgets, and enforce limits without installing the Python SDK. Node, Go, Ruby, bash — anything that can make an HTTP request.

Starting the stack

# The API starts automatically with the full stack
docker compose up -d

# The API is available at http://localhost:8080
# Interactive docs: http://localhost:8080/docs
curl http://localhost:8080/health/live

Authentication

Set KOSTRACK_API_KEY in your .env file to enable bearer token auth. Leave it blank for open access (appropriate for local or private network deployments).

# .env
KOSTRACK_API_KEY=your-secret-token

# All requests (except /health/live)
curl -H "Authorization: Bearer your-secret-token" http://localhost:8080/spend

Endpoints

MethodPathDescription
GET/health/liveLiveness probe — no auth, no DB required
GET/healthWriter health for all connected services
GET/spendSpend by provider/model/project with filters
GET/spend/trace/{trace_id}Full per-model breakdown for one workflow trace
GET/budgetsAll budgets with live period spend and status
POST/budgetsCreate or update a budget
DELETE/budgets/{id}Delete a budget by numeric ID
POST/budgets/checkPre-call budget enforcement (returns 402 if exceeded)
GET/modelsAll models seen with lifetime call count and cost
GET/pricingActive pricing for all known models
POST/pricing/syncSync bundled pricing into TimescaleDB
POST/pricingAdd or update pricing for a model

Budget enforcement — the key use case

POST /budgets/check is designed for pre-call enforcement from any language. Send your tags before making an LLM call. Get back budget states. If a hard budget (enforce: true) is exceeded, the API returns 402 Payment Required — your service should treat this as a signal to abort the call.

# Node.js example
const res = await fetch('http://kostrack:8080/budgets/check', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer ' + process.env.KOSTRACK_API_KEY,
  },
  body: JSON.stringify({
    tags: { project: 'openmanagr', team: 'engineering' },
    estimated_cost: 0.01,
  }),
});

if (res.status === 402) {
  const detail = await res.json();
  throw new Error(`Budget exceeded: ${detail.detail.tag_value} ${detail.detail.period}`);
}

# Proceed with LLM call only if 200
const response = await anthropic.messages.create(...);

OpenAPI docs

The full OpenAPI schema is available at http://localhost:8080/docs (Swagger UI) or http://localhost:8080/redoc. Every endpoint, parameter, and response model is documented there — useful for team onboarding and generating client SDKs.

Production deployment

For production, place the API behind a reverse proxy (nginx or Caddy) and set KOSTRACK_API_KEY to a strong secret. The API container does not handle TLS — terminate SSL at the proxy layer.
