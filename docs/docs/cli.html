CLI Reference

Check spend, set budgets, and inspect AI usage from the terminal — no Python knowledge required. If a developer has installed kostrack, you can govern AI spend with simple commands.

Who this is for

You do not need to know Python to use the CLI. If you can open a terminal and type commands, you can check spend, set budget limits, and inspect AI usage for your whole team. A developer needs to install kostrack once — after that, this is yours.

Installation (one time, done by a developer)

The CLI is bundled with the Python SDK. After one installation command, the kostrack command works anywhere in the terminal:

pip install kostrack

Verify it worked:

kostrack --help

Connecting to your data

The CLI reads from the same database that collects all the AI usage data. You need a connection string (DSN) — your developer can give you this. It looks like:

postgresql://kostrack:yourpassword@localhost/kostrack

Set it once in your terminal session so you never have to type it again:

export KOSTRACK_DSN=postgresql://kostrack:yourpassword@localhost/kostrack

Or pass it directly to any command:

kostrack status --dsn postgresql://kostrack:yourpassword@localhost/kostrack
Argument order

--dsn must come after the subcommand: kostrack status --dsn ... — not kostrack --dsn ... status.

Daily use — checking spend

What did we spend in the last 24 hours?

kostrack status

Shows a table of spend by provider (Anthropic, OpenAI, Gemini, DeepSeek) for the past 24 hours — call count, cost, and average response time.

How much has a specific project spent this month?

kostrack spend --project openmanagr --days 30

Replace openmanagr with your project name. The --days flag sets how far back to look.

Which AI models is the team using?

kostrack models

Lists every AI model that has ever been used, with total call count and total cost — sorted by spend, most expensive first.

What did a specific workflow run cost?

kostrack traces --project openmanagr

Shows recent multi-step AI workflow runs — each row is one workflow execution, with how many AI calls it made, how long it took, and what it cost in total.

Managing budgets

Budgets let you set spending limits on any project, team, or dimension. You can choose a soft limit (alert only) or a hard limit (blocks calls once exceeded).

See all current budgets and their status

kostrack budgets

Shows each budget with a progress bar: how much has been spent, the limit, and whether it is on track, alerting, or exceeded.

Set a monthly budget for a project

# Soft limit — get an alert at 80%, but calls are never blocked
kostrack budget set project openmanagr monthly 50.00

# Hard limit — calls are blocked once $50 is reached
kostrack budget set project openmanagr monthly 50.00 --enforce

# Alert earlier — warn at 70% instead of 80%
kostrack budget set project openmanagr monthly 50.00 --alert 0.70

The four parts are: dimension (project), value (openmanagr), period (monthly), limit in USD (50.00). You can also budget by team, feature, or any tag your developers use.

PeriodResets
dailyEvery day at midnight UTC
weeklyEvery Monday at midnight UTC
monthlyFirst day of each month

Remove a budget

kostrack budget delete project openmanagr monthly

Pricing

Kostrack tracks the cost of every AI call. These commands let you see and manage the pricing data it uses.

See current AI model prices

kostrack pricing

Shows the price per million tokens for every model across all four providers, including cache and batch rates.

Update prices after a provider changes their rates

kostrack pricing sync

Pushes the latest bundled prices into the database. Safe to run at any time — it skips entries that are already up to date.

System health

kostrack health

Shows the status of each service that is writing data to kostrack — whether it is connected, how many records are queued, and when it last wrote successfully.

Getting data out as JSON

Every command supports a --json flag that outputs raw data instead of formatted tables. This is useful for piping into other tools or importing into a spreadsheet:

# Export last 30 days of spend to a file
kostrack spend --days 30 --json > spend.json

# Export all budgets
kostrack budgets --json > budgets.json
