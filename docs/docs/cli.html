CLI Reference

The kostrack command gives operators full cost governance from the terminal — no Python required. Check spend, manage budgets, sync pricing, and inspect traces without writing a line of code.

Installation

The CLI is included with the SDK package. After pip install kostrack, the kostrack command is available globally:

pip install kostrack
kostrack --help

Authentication

All commands require a TimescaleDB connection. Set it once in your shell environment or pass it per-command:

# Set once
export KOSTRACK_DSN=postgresql://kostrack:password@localhost/kostrack

# Or pass per-command
kostrack status --dsn postgresql://kostrack:password@localhost/kostrack
Argument order

--dsn must come after the subcommand name: kostrack status --dsn ..., not kostrack --dsn ... status. This follows standard argparse subparser conventions.

Commands

kostrack status

24-hour spend summary grouped by provider. Good for a quick morning check.

kostrack status
kostrack status --json

kostrack spend

Detailed spend breakdown by day, model, and project. Supports filters and a configurable lookback window.

kostrack spend
kostrack spend --project openmanagr
kostrack spend --project openmanagr --days 7
kostrack spend --provider deepseek --days 30
kostrack spend --json | jq '.[].cost_usd'

kostrack budgets

Show all configured budgets with live spend, a progress bar, and status (ok / alert / exceeded).

kostrack budgets
kostrack budgets --json

kostrack budget set

Create or update a budget. Upserts on the combination of tag key + value + period.

# Soft budget — alert at 80%, never block
kostrack budget set project openmanagr monthly 50.00

# Hard budget — raise BudgetExceededError at 100%
kostrack budget set project openmanagr monthly 50.00 --enforce

# Custom alert threshold
kostrack budget set team engineering monthly 200.00 --alert 0.70 --enforce
ArgumentDescription
tag_keyTag dimension to scope by, e.g. project, team, user_id
tag_valueValue to match, e.g. openmanagr
perioddaily, weekly, or monthly
limit_usdBudget cap in USD
--alertAlert threshold 0–1 (default: 0.80)
--enforceHard limit — SDK raises BudgetExceededError at 100%

kostrack budget delete

kostrack budget delete project openmanagr monthly

kostrack pricing

Show the currently active pricing table — all providers, models, and rates.

kostrack pricing
kostrack pricing --json

kostrack pricing sync

Push the bundled pricing table (current as of the installed SDK version) into TimescaleDB. Idempotent — skips entries that already have identical rates.

kostrack pricing sync

kostrack pricing add

Add or update pricing for a model. Rates are in USD per million tokens. Creates a new versioned entry in the pricing table; previous rates are preserved with effective_to set.

# Add a new model
kostrack pricing add deepseek deepseek-v4 0.30 1.40

# With cache read rate
kostrack pricing add deepseek deepseek-v4 0.30 1.40 --cache-read 0.07

kostrack models

All models ever seen across all services, sorted by total lifetime spend.

kostrack models
kostrack models --json

kostrack traces

Recent agentic workflow traces — each row is one workflow run with call count, duration, and total cost.

kostrack traces
kostrack traces --project openmanagr --limit 50
kostrack traces --json | jq '.[0]'

kostrack health

Writer health for all services that have connected to this TimescaleDB instance.

kostrack health
kostrack health --json

Global flags

FlagDescription
--dsnTimescaleDB DSN. Defaults to $KOSTRACK_DSN.
--jsonOutput raw JSON instead of formatted tables. Useful for scripting.
--serviceFilter output by service_id.
